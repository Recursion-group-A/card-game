# Card Game

# Description

チームで開発した、トランプゲームを遊ぶことができるウェブアプリケーションです。</br>
要件定義、設計、技術選定から実装まで行いました。実装ゲームは、ブラックジャック、ポーカーです。

# Install

```
git clone https://github.com/Recursion-group-A/card-game.git
cd card-game
npm i
npm run dev
```
